# 任务管理网站

###子组件功能
| 名称	    | 功能                        |
| :----------  |:------------------------ |
| Navmen    | 目录显示                    |
| MyList	    | 列表显示                 |
| MyForm	| 添加或修改任务               
| MyCalenda  | 日历                       |
| AllList	    | 过滤条件，数据传给MyList|
